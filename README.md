# Single page Web e-commerce to impelement modern javascript's framework.

Stack technologies:

Back-end :

    Node.js
    Express.js
    Postgresql
    Sequelize-ORM
    Restful API

Front-end:

    React.js
    Redux
    Redux-saga
    React-router
    react-infinite-scroll-component
    react-star-ratings
    Bootsrap 4.5
    SweetAlert2
